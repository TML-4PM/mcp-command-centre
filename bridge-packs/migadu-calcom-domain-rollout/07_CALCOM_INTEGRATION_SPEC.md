# Cal.com Integration Spec

Cal.com is the standard scheduling layer for all in-scope domains.

## Required
- branded org/team per domain
- event types configured
- webhook to bridge
- notifications enabled

## Events
- booking.created
- booking.rescheduled
- booking.cancelled

## Actions
- CRM interaction
- support routing
- finance (if paid)

## Notes
- augmentedhumanity.coach: preserve existing data
- holo-org.com: product bookings
- workfamilyai.org: team routing
